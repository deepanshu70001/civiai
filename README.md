# CiviAI

CiviAI is an AI-powered civic complaint platform that allows users to upload issue images, classify the issue using AI, store complaints, track status, and verify resolution.

## Planned Stack

- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: PostgreSQL + Prisma
- AI: OpenAI Vision
- Image Upload: Cloudinary
