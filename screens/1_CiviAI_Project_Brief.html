CiviAI: An AI-powered civic issue reporting platform. Core User Flows: 1. Report Issue: Image upload, location, AI classification (type, severity, confidence), tracking. 2. Operations Dashboard: Filterable, paginated list of complaints with quick drill-down. 3. Complaint Detail: Full info, evidence, status updates, after-image verification, audit timeline. 4. Admin Queue: Bulk operational view, fast updates, status counts. 5. Impact Center: Executive metrics (total, resolution rate, etc.), hotspots, priority queue, "Civic Impact Pulse" narrative summary. Design Direction: - Premium civic-tech feel: clean, trustworthy, data-first. - Focus on clarity for citizens and officials. - Impact Center should be the visually strongest "hero" screen. - Mobile responsive and fast readability. UI Components to Include: - App shell/top nav - Cards/panels - Forms (inputs, selects, textarea, file upload) - Buttons (primary/secondary/ghost) - Status/severity/issue pills - Skeleton loaders - Metric stat chips - List items + timeline blocks - Notice banners - Responsive layouts + smooth animations/stagger transitions.
